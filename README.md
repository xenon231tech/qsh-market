# 🖼️ QSH Market

Marketplace NFT untuk mint, list, dan beli NFT dengan mudah

## 🌟 Features

- ✨ Modern, responsive design
- 🎨 Beautiful UI with glassmorphism effects
- 🚀 High performance optimization
- 📱 Mobile-first approach
- ♿ Accessibility compliant (WCAG)
- 🔒 Security best practices
- ⚡ Fast loading times
- 🌙 Dark/Light mode support

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://xenonhub-kfbaaxzf.manus.space/qsh-market.git
   cd qsh-market
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

## 🎨 Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    --accent: #00d4ff;
}
```

### Content
Edit the HTML content in `index.html` to customize:
- Title and subtitle
- Description
- Features
- Links

## 🌐 Deployment

### GitHub Pages
1. Go to repository Settings → Pages
2. Select "main" branch as source
3. Website will be live at: `https://xenonhub-kfbaaxzf.manus.space/qsh-market`

### Vercel
1. Connect repository to Vercel
2. Deploy automatically
3. Website will be live at: `https://qsh-market.vercel.app`

### Netlify
1. Connect repository to Netlify
2. Deploy automatically
3. Website will be live at: `https://qsh-market.netlify.app`
